# Skoggy.Grove

Classlib for MonoGame

## Working on

* nothin' and everything

## Features Implemented

* Animations (Frame based)
* Timers
* Poolable collection
* Generic FSM
* Sprite Sheets
* Tiled support (TMX maps)
* Scene Management
* Scene Graph (via entity system)
* Entity Components
* Multi-layer rendering support (for sprites)
* Nuget package
